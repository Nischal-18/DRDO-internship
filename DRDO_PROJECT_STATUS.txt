PROJECT OVERVIEW
================================================================================
A web portal for DRDO where:
- Users can register, login, view dashboard, and apply for programs
- Admins can login, view all applications, manage users, and approve/reject applications

Tech Stack:
- Frontend: React 19 + TypeScript + Tailwind CSS 4 + Vite 7
- Backend: Node.js + Express
- Database: Oracle XE (Docker container: oracle-free)
- Authentication: JWT tokens

BACKEND APIs (All Functional)
--------------------------------------------------------------------------------
| Method | Endpoint                        | Description              | Auth    |
|--------|--------------------------------|--------------------------|---------|

| PUT    | /api/users/:id/role            | Update user role         | Admin   |
| PUT    | /api/users/:id/status          | Activate/deactivate user | Admin   |

   File: /DRDO-frontend/src/context/AuthContext.tsx
   Changes:
   - On app load, if token exists in localStorage:
     * Calls GET /api/auth/profile to verify token
     * If valid: refreshes user data from backend
     * If invalid: clears token and logs user out
   - Prevents users with expired/invalid tokens from staying logged in

   File: /DRDO-backend/schema.sql
